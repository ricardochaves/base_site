A skeleton with Django and Docker

## Execute

Clone the project

```
git clone https://github.com/ricardochaves/base_site.git
```

Go to `base_site` dir

```
cd base_site
```

Execute the `docker-compose.yml`

```
docker-compose up
```

Acess `localhost:5005`
